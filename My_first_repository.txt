КОНТРОЛЬНІ ЗАПИТАННЯ:
1. GIT - це система керування версіями та спільної роботию
2. - змінені файли, добавлені файли, чекоют коміту та не добавлені файли.
3. Add - добавляє файли на стадію коміту.
4. Status - показує стан файлів.
5. Commit - збирає всі данні добавленні командою git add i зберігає їх.
6. Init - перетворює звичайну директорію в Git репозиторій.
7. - опис загальних змін які відбулися.
8. - щоб переглянути список комітів потрібно написати git long -- since// за який час
9. Checkout - використовується для переключення між гілками та вигрузки їх складу в робочу директорію
